GitHub Bağlantısız

GitHub Personal Access Token

GitHub Settings > Developer settings > Personal access tokens > Generate new token
Gerekli izinler: repo (tüm repository erişimi)

GitHub Token Nasıl Oluşturulur?

  1. GitHub.com'da giriş yapın
  2. Settings > Developer settings > Personal access tokens'a gidin
  3. "Generate new token" butonuna tıklayın
  4. Token'a bir isim verin (örn: "Website Admin Panel")
  5. repo iznini seçin (tüm repository izinleri)
  6. "Generate token" butonuna tıklayın
  7. Oluşan token'ı kopyalayın ve yukarıdaki alana yapıştırın

Başarılı!
